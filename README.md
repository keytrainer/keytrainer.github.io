# Touch Typing Trainer

A simple, vanilla web application that teaches correct touch typing with proper finger placement on the keyboard.

## Features

- **Visual Keyboard:** On-screen color-coded keyboard showing correct finger placement.
- **Finger Guidance:** Highlights keys and shows which finger should press it.
- **Progressive Lessons:** Lessons ranging from basic home row to full alphabet.
- **Real-Time Feedback:** Highlights for correct and incorrect keystrokes.
- **Statistics:** Real-time Words Per Minute (WPM), accuracy, and mistake counting.

## Setup

No build tools or frameworks are required. You can either:
1. Open `index.html` directly in your browser.
2. Host it on any static file server like GitHub Pages.

## Tech Stack

- Vanilla HTML
- Vanilla CSS
- Vanilla JavaScript
